
### Getting Started

Requirements
 - Node.js


#### 1. Clone the repo

```sh
git clone https://github.com/hawyar/ferg-layout-challenge.git
```

####  2. Install dependencies


```sh
npm install
# or yarn
```

####  3. Run

```sh
npm run dev
# yarn dev
```

Or to watch
```sh
npm run watch
# yarn watch
```
